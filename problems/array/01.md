## Solução
```cpp
class Solution {
public:
    vector<int> twoSum(vector<int>& nums, int target) {
        int id1{}, id2{};
        int sum = nums.at(0);
        for(int i{1}; i < nums.size(); i++){
            auto it = find(nums.begin()+i, nums.end(), (target-sum));

            if(it != nums.end()){
                id2 = distance(nums.begin(), it);
                break;
            }else{
                id1 = i;
                sum = nums.at(i);
            }
        }
        vector<int> indices{id1,id2};
        return indices;
    }
};

```

---

## Entendendo o problema e solução

Dado um array de inteiros **nums** e um inteiro **target**, é preciso retornar os índices dos dois números cuja soma seja igual a **target**.

Simples!

- Declarei os inteiros **id1** e **id2** para guardar os indices dos números cuja soma é igual a **target**;
- O loop será necessário para iterar pelo array **nums**, tendo inicio no segundo indice;
- Anteriormente declarei o inteiro **num**, inicializado com o primeiro elemento de **nums**, porém, ele sempre conterá o elemento do indice anterior ao indice atual no loop. Ele também irá sinalizar o indice do primeiro número que buscamos;
- Agora Declarei um auto **it** que me dará o local do segundo número se esta iteração for a correta e, fazendo uso da função **find**, analisamos o array a partir de **i** passando como parametro (**target**-**sum**) que é exatamente o segundo número que precisamos.
- Se a função **find** encontrar o número esperado segundo a iteração, fazendo o uso da função **distance** finalizarei a busca do segundo indice, uma vez que, o primeiro é (i-1), então encerro o loop. Caso contrário continuo até cair no if;
- Por fim, crio um vector do tipo int inicializado com **id1** e **id2**, em seguida retorno este vector contendo os indices dos dois números cuja soma é igual a **target**.
