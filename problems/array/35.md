## Problema
Categoria: Array

Problema: 35.Search Insert Position

Nível de dificuldade: Fácil

Link: https://leetcode.com/problems/search-insert-position/description

---

## Solução
```cpp
class Solution {
public:
    int searchInsert(vector<int>& nums, int target) {
        int indice{};
        auto it = find(nums.begin(), nums.end(), target);
        if(it != nums.end()){
            indice = distance(nums.begin(), it);
        }else{
            nums.push_back(target);
            sort(nums.begin(), nums.end());
            it = find(nums.begin(), nums.end(), target);
            indice = distance(nums.begin(), it);
        }
        return indice;
    }
};

```

---

## Entendendo o problema e solução

Dado um array **nums** ordenado de inteiros distintos e um valor alvo **target**, devemos retornar o índice se o valor alvo for encontrado. Caso contrário, retornar o índice onde ele estaria se fosse inserido na ordem correta.

- Primeiro declarei um inteiro **indice** que guardará o indice de **target** em **nums**;
- Agora um auto **it** que receberá a localização de **target** a partir da função **find**, responsável pela busca;
- Se **target** se encontrar em **nums**, fazendo uso da função **distance**, iremos atribuir a distancia entre o inicio de **nums** e **it** à indice. Caso contrário, se **target** não estiver em **nums**, usaremos a funcão **push_back** para adicioná-lo, usaremos **sort** para ordenar **nums** novamente e jogar **target** no lugar certo, então, faremos a busca novamente usando **find** e **distance** para encontar seu indice;
- Por fim, retornamos indice.
