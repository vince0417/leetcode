## Problema
Categoria: String

Problema: 28.Find the Index of the First Occurrence in a String

Nível de dificuldade: Fácil

Link: https://leetcode.com/problems/find-the-index-of-the-first-occurrence-in-a-string/description

---

## Solução
```cpp
class Solution {
public:
    int strStr(string haystack, string needle) {
        auto it = haystack.find(needle);
        if (haystack.find(needle) == string::npos) return -1;
        else return it;
    }
};

```

---

## Entendendo o problema e solução

Dadas duas strings, **needle** e **haystack**, devemos retornar o índice da primeira ocorrência de **needle** em **haystack**, ou -1 se **needle** não fizer parte de **haystack**.

Este é um problema bem simples de resolver. Primeiro, foi criado um **it** que receberá o índice da primeira ocorrência de **needle** em **haystack**, caso ela esteja lá. Em seguida verificamos se **needle** está em **haystack** se não retornamos -1 e finalizamos, caso contrário, se estiver lá, retornamos **it**, seu local de início.
