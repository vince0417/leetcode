## Problema
Categoria: String

Problema: 557.Reverse Words in a String III

Nível de dificuldade: Fácil

Link: https://leetcode.com/problems/reverse-words-in-a-string-iii/description

---

## Solução
```cpp
class Solution {
public:
    string reverseWords(string s) {
        stringstream str(s);
        string palavra;
        s.clear();
        while(str >> palavra){
            reverse(palavra.begin(), palavra.end());
            if(s.length() != 0) s += ' ';
            s += palavra;
        }
        return s;
    }
};

```

---

## Entendendo o problema e solução

Dada uma string **s**, preciso inverter a ordem dos caracteres em cada palavra dentro de uma frase, preservando os espaços em branco e a ordem inicial das palavras.
- Priemiro, separei as palavras da frase usando um objeto **stringstream** contido na biblioteca *sstream* inicializado com a string **s**;
- Criei uma nova string para receber cada palavra da frase;
- Limpei a string **s** usando .clear() para receber a frase desta vez com as palavras invertidas;
- Agora é hora do while para percorrer o objeto e, fazendo uso do operador **>>** que usa espaços em branco (espaços, tabs, novas linhas) como delimitador, entre uma palavra e outra, fiz a extração de cada palavra, atribuindo a mesma a string **palavra** criada anteriormente. Obs: a cada iteração do while é retirada uma palavra do objeto, até que não reste mais e o loop seja encerrado;
- Em cada iteração do loop, fazendo uso da função **reverse** definida pelo cabeçalho *algorithm*, inverto **palavra**, concateno um espaço ' ' e concateno a palavra, assim, refazendo a frase;
- Por fim, retorno **s**.
