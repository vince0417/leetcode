## Problema
Categoria: String

Problema: 58.Length of Last Word

Nível de dificuldade: Fácil

Link: https://leetcode.com/problems/length-of-last-word/description

## Solução
```cpp
class Solution {
public:
    int lengthOfLastWord(string s) {
        reverse(s.begin(), s.end());
        int aux{}, wordLength{};
        for(size_t it{}; it < s.length(); ++it){
            if(aux == 1 && s.at(it) == ' ') break;
            if(s.at(it) != ' '){
                aux = 1;
                ++wordLength;
            }

        }
        return wordLength;
    }
};

```

---

## Entendendo o problema e solução

Dada uma string **s** composta por palavras e espaços, devemos retornar o comprimento da última palavra na string.

Muito simples!
- Para isso, primeiro, usei a função **reverse** definida pelo cabeçalho **algorithm** que inverte início e fim da string. Assim, irá facilitar a busca pela última palavra e seu respectivo tamanho. 
- Após isso, declarei duas variáveis do tipo inteiro, **aux** que será usada como condição para encerrar o for a seguir, e **wordLength** que guardará o tamanho da palavra.
- Por último criei um for para percorrer a string **s**, com dois **if´s** o primeiro é nossa condição para encerrar o loop, após encontrar a palavra. O segundo é o que conta  o tamanho da nossa palavra e garante 1 à **aux** que é uma das nossas condições de finalização.
- Agora retornamos **wordLength**, que contém o comprimento da nossa palavra.
