## Problema
Categoria: String

Problema: 771.Jewels and Stones

Nível de dificuldade: Fácil

Link: https://leetcode.com/problems/jewels-and-stones/description

---

## Solução
```cpp
class Solution {
public:
    int numJewelsInStones(string jewels, string stones) {
        int myJewel{};
        for(size_t it{}; it < stones.length(); ++it){
            if(jewels.find(stones.at(it)) != string::npos) myJewel++;
        }
        return myJewel;
    }
};

```

---

## Entendendo o problema e solução

Você recebe duas strings **jewels** e **stones** representando, respectivamente, os tipos de pedras que são joias e as pedras que você possui. Cada caractere em "stones" representa um tipo de pedra que você tem. Você quer saber quantas das pedras que você possui também são joias.

As letras diferenciam maiúsculas de minúsculas, então "a" é considerado um tipo de pedra diferente de "A".

Simples! 
- Primeiro criei uma variável do tipo **int** chamada **myJewel** que receberá a quantidade de jóias que temos;
- Criei um loop **for** para percorrer minhas pedras, a cada iteração verifico se aquele caractere está presente em **jewels**, se sim, icremento **myJewel**;
- Por fim, retorno **myJewel**.
